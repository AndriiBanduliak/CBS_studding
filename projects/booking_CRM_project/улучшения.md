# Идеи улучшений и новые фичи

## Архитектура и инфраструктура
- Перейти на PostgreSQL для продакшна (транзакции, индексы, расширения)
- Docker/Docker Compose для быстрого онбординга и единообразия окружений
- Добавить requirements.txt/poetry + pip-compile (lock-файлы)
- CI (GitHub Actions): линт, тесты, сборка фронтенда, безопасность (pip-audit)
- Разделить settings.py на base/dev/stage/prod, убрать дефолтные секреты

## Безопасность и конфигурация
- В проде выключить CORS_ALLOW_ALL_ORIGINS, настраивать по списку
- Хранить секреты в .env/секретах CI, добавить строгий .env.example
- HTTP security headers (SecurityMiddleware + CSP)
- Sentry/Errors tracking, структурированные логи (JSON) + log rotation

## Бэкенд (Django + DRF)
- Добавить JWT/Session Hardened аутентификацию, refresh-токены, ротирование
- Ролевая модель и объектные пермишены (Django Guardian/кастомная RBAC)
- Расширить фильтры/поиск (django-filter), индексы БД на частые поля
- Асинхронные задачи (Celery + Redis) для уведомлений/синков календаря
- Вебхуки Google: подпись, ретраи, idempotency ключи
- Версионирование API, стабильные схемы и чёткие коды ошибок
- Генерация SDK/клиента из OpenAPI, автотесты контрактов

## Данные и доменная логика
- Валидация пересечений бронирований + подбор альтернатив
- Календарная доступность с агрегированием по объектам и сегментам дат
- Динамическое ценообразование (сезоны, промо, occupancy-based rules)
- История изменений (audit trail уже есть — расширить диффами и актёрами)

## Тестирование и качество
- Единицы измерения покрытия: unit + integration + API контрактные тесты
- Factory Boy/Faker для сидов тестовых данных
- Линтеры/форматеры: ruff/flake8 + isort + black; pre-commit hooks
- Миграции: тест «migrate up/down» на чистой БД в CI

## Фронтенд (React + Vite + AntD)
- Состояние: Zustand/Redux Toolkit Query для кэширования API
- Роут-гварды, приватные маршруты, перехват 401/403
- Error Boundary + централизованные уведомления об ошибках Axios
- Скелетоны/лоадеры и оптимистичные апдейты
- Табличные списки: сохранение состояний фильтров/сортировки/страницы в URL
- Мобильные экраны: задачи/календарь, offline-first для чеклистов
- E2E (Playwright) + визуальные регресс-тесты критичных экранов

## Отчётность и аналитика
- Отчёты по загрузке, выручке, каналам продаж, отменам
- Экспорт CSV/XLSX, планировщик рассылки отчётов
- Дашборды (например, Metabase/Superset) поверх read-only replica

## Интеграции
- Google Calendar двусторонняя синхронизация, дедупликация событий
- Каналы бронирований (Airbnb/Booking.com) — агрегатор и нормализация данных
- Платёжные провайдеры (Stripe/LiqPay/WayForPay)

- Makefile/Taskfile для типовых команд (run, test, lint, migrate)
